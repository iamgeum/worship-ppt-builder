- React/Vite 기반 프론트엔드 스캐폴딩
- Agent 1 메뉴: 좌측 고정 메뉴, 접기/펼치기, 활성 메뉴 표시, 악보 편집 준비중
- Agent 2 개인화: 스타일 프리셋, 비율, 여백 조정, 되돌리기
- Agent 3 순서 관리: 슬라이드 카드 목록, 추가, 삭제 확인, 드래그 재정렬, 빈 슬라이드 감지 모달
- Agent 4 편집기: 슬라이드 비율, 여백 점선, 캐러셀 스와이프, 하단 5개 미리보기, 텍스트/도형/이미지 요소 추가, 요소 이동
- Agent 6 텍스트 에디터: 엔터 2번 기준 페이지 분할, 생성 페이지 수 표시, 미리보기, 저장 시 슬라이드 생성, 간단한 말씀 단축 검색
- Agent 7 UX 벤치마킹: URL 입력과 디자인 토큰 미리보기 UI
- `localStorage` 자동 보존 및 전역 undo 히스토리
- 빌드 오류 수정: JSX가 포함된 Context 파일을 `src/context/WorshipContext.jsx`로 정리
- 검증: `npm.cmd run build` 성공
- 검증: Vite dev 서버 `http://127.0.0.1:5173` 응답 200 확인
- 검증: 인앱 브라우저에서 첫 화면과 편집 화면 주요 문구 확인

### 2026-05-22 편집기 조작 개선

- 이미지 파일 업로드 버튼을 추가했습니다.
- 업로드된 이미지는 `data:image/...` 형태로 슬라이드 상태에 저장되어 PPTX export에 포함될 수 있습니다.
- 선택한 텍스트/도형/이미지 요소에 4개 모서리 리사이즈 핸들을 표시합니다.
- 리사이즈 중 요소의 x/y/w/h 퍼센트 좌표를 갱신하고 슬라이드 경계 밖으로 나가지 않게 제한합니다.
- 이미지가 들어간 경우 기본 URL 입력창은 숨기고, 선택/호버/포커스 시 다시 편집할 수 있게 했습니다.
- 검증: `npm.cmd run build` 성공
- 참고: dev 서버 백그라운드 실행이 이번 세션에서 안정적으로 유지되지 않아 브라우저 재검증은 보류

### 2026-05-22 선택 요소 조작 개선

- 선택한 요소를 툴바에서 복사, 붙여넣기, 삭제할 수 있게 했습니다.
- 선택한 요소는 `Ctrl/Cmd+C`, `Ctrl/Cmd+V`, `Delete/Backspace`로도 조작할 수 있습니다.
- 선택한 요소는 방향키로 1% 단위 이동, `Shift+방향키`로 5% 단위 이동할 수 있습니다.
- 텍스트 입력창, 이미지 URL 입력창, select 편집 중에는 단축키를 가로채지 않도록 처리했습니다.
- 붙여넣기 시 새 요소 id를 부여하고 살짝 오프셋해 겹침을 알아볼 수 있게 했습니다.
- 검증: `npm.cmd run build` 성공

### 2026-05-22 레이어/정렬/snapping 개선

- 선택 요소를 앞으로/뒤로 보내는 레이어 순서 버튼을 추가했습니다.

- 드래그 snapping을 슬라이드 경계/중앙뿐 아니라 다른 요소의 시작/끝/중앙 기준선까지 확장했습니다.
- snapping 후에도 요소가 슬라이드 경계를 벗어나지 않도록 좌표를 제한합니다.
- 검증: `npm.cmd run build` 성공
