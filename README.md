# ⛪ 예배 PPT 빌더 (Worship PPT Builder)

> 어르신도, 아이도 쉽게 — 누구나 5분 안에 예배 PPT를 만들 수 있는 웹 애플리케이션

---

## 🎯 이게 뭐하는 앱인가요?

교회 예배를 준비할 때 찬양 가사, 성경 말씀, 광고 등을 PPT로 만드는 작업은 매번 반복되고 번거롭습니다.  
이 앱은 그 과정을 최대한 단순하게 만들어, **컴퓨터에 익숙하지 않은 분도 클릭 몇 번으로 예배 PPT를 완성**할 수 있도록 설계되었습니다.

### 핵심 특징

- 드래그&드롭으로 슬라이드 순서 조정
- 가사 붙여넣기 → 자동으로 슬라이드 분할
- 성경 말씀 역본/권/장/절 선택 또는 단축 검색 (`창 1:1-10`)으로 즉시 불러오기
- PPT와 동일한 캐러셀 방식 편집 화면
- 기본값 포맷 제공 — 수정 없이 바로 사용 가능

---

## 🗺️ 개발 방향

### 기본 원칙

| 원칙 | 설명 |
|---|---|
| **클릭 수 최소화** | 모든 기능은 3번 이내 클릭으로 도달 |
| **실수해도 괜찮음** | 되돌리기 버튼 항상 노출, 삭제 전 확인 팝업 |
| **지금 뭘 하는지 항상 보임** | 현재 위치, 저장 상태, 페이지 수 실시간 표시 |
| **기본값으로 바로 사용 가능** | 포맷 수정 없이도 완성된 PPT 출력 가능 |

### 개발 단계

```
1단계 (MVP)
├── 메뉴 구조 및 화면 라우팅
├── 순서페이지 관리 (슬라이드 추가/삭제/정렬)
├── 가사/말씀 텍스트 에디터
└── 기본 편집 에이전트 (캐러셀 방식)

2단계
├── 로그인 및 개인 포맷 저장
├── 슬라이드 개인화 에디터 (노코딩 드래그 방식)
└── PPT 내보내기

3단계 (보류)
└── 악보 이미지 편집 에디터
```

## ⚙️ 공용 컴포넌트

### SlidePreviewModal (가사/말씀 공통 사용)
- 흰 배경 + 검정 텍스트
- 좌우 화살표로 페이지 이동
- 현재 N / 전체 M 페이지 표시
- 우측 상단 X 버튼으로 닫기

### WorshipContext (전역 상태)
모든 에이전트가 하나의 상태를 구독  
→ 편집 에이전트에서 수정한 내용이 순서 관리 탭 썸네일에 실시간 반영

## 🔖 용어 정리

| 용어 | 설명 |
|---|---|
| 슬라이드 | PPT 한 장 |
| 순서페이지 | 예배 순서를 슬라이드 목록으로 관리하는 화면 |
| 페이지 나뉨 | 가사/말씀 텍스트에서 엔터 2번으로 생기는 슬라이드 구분선 |
| 캐러셀 편집 | 슬라이드를 위아래가 아닌 좌우로 이동하며 편집하는 방식 |
| 역본 | 성경 번역본 (예: 개역개정, 쉬운성경 등) |
