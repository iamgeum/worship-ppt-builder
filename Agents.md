## 에이전트 1 — 메뉴 에이전트

**폴더:** `src/agents/01_menu/`  
**역할:** 앱 전체의 네비게이션을 담당. 좌측 고정 메뉴로 각 에이전트 화면에 연결.

### 기능 목록

- 페이지 가장 좌측에 세로 메뉴 고정 배치
- 메뉴 항목: 아이콘 + 한글 레이블 **동시 표시** (예: "순서 짜기", "PPT 만들기")
- 현재 활성 화면 메뉴 항목을 색상/굵기로 강조
- 편집 화면에서 작업 공간 확보를 위해 메뉴 **접기/펼치기** 지원 (아이콘만 표시)
- 좌측 하단에 **"나가기"** 버튼 고정 노출
- 추후 순서페이지 에이전트(3번)와 연결될 수 있도록 라우팅 구조 확장 가능하게 설계

### UX 유의사항

- 모든 메뉴 버튼 최소 **44px 이상** (터치/클릭 정확도)
- 아이콘만으로 기능 파악이 어려운 사용자 고려 → 레이블 숨김 불가 (기본 설정)
- 첫 방문 시 각 버튼 기능 안내 툴팁 또는 튜토리얼 흐름 연결 가능하도록 구조 예비

## 에이전트 2 — 슬라이드 개인화 에이전트

**폴더:** `src/agents/02_personalizer/`  
**역할:** 슬라이드 기본 포맷(스타일, 여백, 폰트 등)을 설정하고 개인화 가능하게 관리.

### 기능 목록

- 슬라이드별 기본값 제공 → 수정 없이 그대로 사용 가능
- 스타일 프리셋 제공 (예: "기본 흰 배경", "어두운 배경", "사진 배경" 등 3~5종)
- 노코딩 드래그 방식 편집 (마우스로 요소를 자유롭게 이동)
- Snapping(자석 정렬): 요소가 중앙선/모서리에 근접 시 자동 정렬 가이드라인 표시
- **되돌리기(↩)** 버튼 에디터 상단 항상 노출
- 로그인 기능 추가 후: 개인별 포맷 저장 및 불러오기 지원 (2단계)

### UX 유의사항

- 로그인 전에도 `localStorage`로 마지막 편집 상태 자동 보존
- 재방문 시 "이전에 만들던 거 이어서 하시겠어요?" 팝업

## 에이전트 3 — 순서페이지 관리 에이전트

**폴더:** `src/agents/03_order-manager/`  
**역할:** 예배 순서를 슬라이드 목록으로 관리. 추가/삭제/순서변경 담당.

### 기능 목록

- 슬라이드 목록을 카드 형태로 나열
- 각 슬라이드 박스 안에 **내용 미리보기 썸네일** 표시 (PPT 좌측 패널을 하단으로 이동한 형태)
- **추가 버튼(파랑)** / **삭제 버튼(빨강)**: 플로팅 또는 고정 배치로 스크롤 위치 무관하게 항상 접근 가능
- 슬라이드 박스 **바깥 우측**에 삭제 버튼 추가 배치
- 슬라이드 박스 **안 우측**의 `+` 버튼 → 우측에 편집 에디터(4번) 활성화
- 햄버거 버튼 드래그&드롭으로 슬라이드 순서 변경
- **저장 시 빈 슬라이드 감지 팝업**
  - "비어있는 슬라이드가 있습니다 — 7번, 12번"
  - 숫자 버튼 클릭 시 해당 슬라이드(번호 빠른 순)로 즉시 이동
  - 이동 시 편집 에이전트(4번)가 해당 슬라이드 활성화 상태로 오픈

### UX 유의사항

- 삭제 전 확인 팝업 필수 ("정말 삭제하시겠어요?" + 확인/취소)
- 작업 완료 시 "저장되었습니다 ✓" 토스트 메시지

## 에이전트 4 — 편집 에이전트

**폴더:** `src/agents/04_editor/`  
**역할:** 슬라이드 내용을 캐러셀 방식으로 편집. 텍스트박스, 도형, 이미지 삽입 지원.

### 기능 목록

#### 편집 화면 구조
- 에디터는 **미니 PPT 그 자체** — 슬라이드 바깥 회색 영역 없음
- 슬라이드 비율(16:9 / 4:3)은 **규격설정**에서 지정, 에디터는 해당 비율로 꽉 채움
- 여백 가이드라인: 설정된 여백을 **파란 점선**으로 항상 표시 (출력 시 미표시)

#### 슬라이드 이동 (캐러셀)
- 슬라이드를 클릭 후 좌/우로 드래그 → 화면 이동
- 드래그 속도에 따라 여러 장 이동 (최대 4장)
- 마우스 재클릭 시 이동 멈춤
- **하단 미리보기**: PPT 좌측 패널을 하단으로 이동한 구조
  - 5개 고정 표시
  - 현재 편집 중인 슬라이드 **파란 테두리** 강조
  - 미리보기 클릭 시 해당 슬라이드로 즉시 이동

#### 요소 삽입 및 편집
- 텍스트박스, 도형, 이미지 삽입 가능
- 요소 선택 시 Figma 스타일 핸들 표시 (크기 조절 포인트 + 위치 수치)
- Snapping: 다른 요소 또는 슬라이드 경계에 근접 시 자동 정렬 가이드라인

#### 에디터 바깥 상단 버튼
- **슬라이드 규격 설정** (여백 포함)
- **가사 추가** → 텍스트 에디터(6번) 실행
- **말씀 추가** → 역본/권/장/절 선택 패널 포함한 텍스트 에디터(6번) 실행

### UX 유의사항

- 캐러셀 스와이프 감도: 기본값 우선 구현, 추후 설정에서 조정 가능하도록 구조 예비
- **되돌리기(↩)** 버튼 항상 노출
- 화면 어딘가에 "자동저장됨 14:32" 상시 표시

### 공통 텍스트 에디터 기능

#### 입력 영역
- 기본 노트패드 형태 (txt 파일 추가 또는 직접 붙여넣기)
- **엔터 2번** → `--- 페이지 나뉨 ---` 구분선 실시간 표시
  - 구분선 삭제 시 두 문단 자동 합쳐짐 (1줄로 복원)
  - 엔터 2번 초과(3번, 100번 등) → 저장 시 초과 빈 줄 자동 제거, 구분선 1개만 유지
  - 예: `abc` + 엔터 100번 + `dnd` → 저장 시 abc(1페이지) / dnd(2페이지)

#### 폰트 자동 조정
- 기본 폰트 크기: **22pt**
- 글자 수에 따라 자동 축소
- 최소 폰트 크기: **10pt** (이 이하로는 자동 축소 안 함)
- 10pt로도 내용이 넘칠 경우: 빨간 경고 표시 + "이 페이지 내용이 너무 많습니다. 직접 줄이세요" 안내

#### 상단 상태바
- **"현재 N페이지 생성 예정"** 실시간 카운트

#### 하단 버튼
- **저장** / **미리보기** / **취소**

#### 미리보기 (SlidePreviewModal 공용 컴포넌트)
- 흰 배경 + 검정 텍스트
- 좌우 화살표로 페이지 이동
- 현재 N / 전체 M 페이지 표시
- 우측 상단 X 버튼으로 닫기

### 말씀 추가 전용 패널 (에디터 상단 부착)

#### 탭 구분
- **한 절** 탭: 단일 구절 선택 (장 + 절만 입력)
- **여러 절** 탭: 시작(장/절) ~ 끝(장/절) 범위 선택

#### 선택 흐름
```
[역본 ▼]  →  [권 ▼]  →  [장 ▼]  →  [절 ▼]
(개역개정)     (창세기)    (1장)     (1절)

여러 절 탭:
시작  [장 ▼] [절 ▼]     끝  [장 ▼] [절 ▼]
```
- 권 선택 → 장 드롭다운 활성화 → 절 드롭다운 활성화 (단계적 활성화)
- 시작과 끝은 **장과 절만** 범위 선택 적용 (역본, 권은 동일하게 적용)

#### 단축 검색창 (드롭박스 최하단 배치)
```
[개역개정 ▼]  [창 1:1-10 입력 후 🔍 검색]
```
- 역본 드롭박스는 단축검색창 왼쪽 고정
- 기본값: 개역개정
- `창 1:1`, `요 3:16`, `롬 8:28-39` 형태 인식

#### 말씀 추가 버튼
- **[추가]** 버튼 클릭 → 에디터 텍스트 하단에 append
  - 상단 줄바꿈 1줄 + 말씀 텍스트 + 하단 줄바꿈 1줄 자동 삽입
  - 여러 구절을 반복해서 추가 가능 (신약/구약 번갈아 사용 가능)

#### 최근 사용 구절 히스토리
- 최근 불러온 구절 최대 10개 하단에 표시
- 클릭 시 즉시 재사용

## 성경 DB 구조

**경로:** `src/data/bible/`

```json
// 예시: 개역개정 창세기 1장 1절
{
  "version": "개역개정",
  "book": "창세기",
  "bookCode": "창",
  "chapter": 1,
  "verse": 1,
  "text": "태초에 하나님이 천지를 창조하시니라"
}
```

- 역본별 파일 분리 관리
- 권/장/절 순서 그대로 저장 → 시작~끝 범위를 순서대로 슬라이스하여 불러오는 방식
